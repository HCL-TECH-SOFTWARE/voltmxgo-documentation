 * - "PITCHINSENSITIVE"

 * <pre>["ASCENDING", "CASESENSITIVE", "ACCENTSENSITIVE", "PITCHSENSITIVE"]</pre>

 * specify "DESCENDING" in the sort function. If conflicting keywords are passed, the last one in the list affects the sort order.
